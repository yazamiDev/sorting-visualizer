Sorting Visualizer

Sorting is one of the fundementable building blocks in Computer Science. It is the process of organizing elements of a data set in a particular order. Not only does sorting reduce the complexity of a problem but also makes searching data faster. Each sorting algorithm has their own benefits and weak areas. Sorting Visualizer illustrates the effectiveness and performance of various sorting algorithm techniques based on their input size and initial condition. Note, the algorithms implemented are their general versions. Some algorithms can be further optimized. Sorting Visualizer goal is to demonstrate how sorting algorithms function. It is important to note that the running time are close approximations. To get started, follow the stepper and select the input size and initial condition on the data set. Once you have your conditions selected, click the "Sort" button to start the visualization!
